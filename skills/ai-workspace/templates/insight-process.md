# Insight Process

Use the user's workspace language for headings, filenames, and path segments.
`insight` is the semantic layer name, not a required English filename.

## Purpose

Create current judgment grounded in evidence, memory, and tension.

Insight answers:

- what does this mean now?
- why should the user care?

## Inputs

Allowed inputs:

- source evidence;
- current or new memory layer file;
- current or new tension layer file;
- existing current insight layer file.

## Rules

- Start from evidence.
- Use memory for durable context.
- Use tension for current pressure.
- Separate confidence from certainty.
- Do not present interpretation as fact.
- Tie the insight to a domain decision.

## Output

```md
## Insight

- ...

## Reasoning

- ...

## Consequence

- ...

## Domain Impact

- ...

## Confidence

High | Medium | Low
```

## TTL

Default TTL: `P7D`.
